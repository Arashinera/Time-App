Build Tools
---------------
Створіть Maven-проект Time-App, де реалізуйте функціонал
переформатування тривалості у секундах (наприклад, 15973 с)
в години, хвилини та секунди за форматом hh:mm:ss.
+
Для переформатування застосуйте бібліотеку Time4J
https://github.com/MenoData/Time4J?tab=readme-ov-file.
+
Забезпечте розподілення функціоналу програми за різними
відповідними класами.
+
